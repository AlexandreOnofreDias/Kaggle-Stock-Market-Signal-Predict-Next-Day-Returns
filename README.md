# Kaggle-Stock-Market-Signal-Predict-Next-Day-Returns
Kaggle competition — Ensemble ML (XGBoost/LightGBM/CatBoost + Optuna) to predict next-day equity returns

Binary classification to predict next-day equity returns.

## Approach
- Feature engineering: vol-adjusted Sharpe ratios, VoV, trend confirmation signals
- Models: XGBoost, LightGBM, CatBoost tuned with Optuna
- OOF ensemble with optimized blending weights

## Status
Work in progress — model under active improvement.

## Stack
Python, XGBoost, LightGBM, CatBoost, Optuna, Scikit-learn
